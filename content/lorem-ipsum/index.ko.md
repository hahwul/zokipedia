# Lorem ipsum이란?

Lorem ipsum은 그래픽 디자인, 출판, 웹 개발 분야에서 널리 사용되는 더미 텍스트(dummy text) 또는 플레이스홀더 텍스트(placeholder text)입니다. 이 텍스트의 주요 목적은 실제 콘텐츠가 채워지기 전에 페이지 레이아웃을 독립적으로 디자인할 수 있도록 하거나, 다양한 글꼴을 시연할 때 의미 있는 텍스트가 디자인 요소에 방해가 되지 않도록 하는 것입니다.

## 역사와 기원

Lorem ipsum은 일반적으로 로마의 정치가이자 철학자인 마르쿠스 툴리우스 키케로(Marcus Tullius Cicero)의 기원전 1세기 저서인 "De finibus bonorum et malorum"의 일부를 기반으로 합니다. 이 텍스트는 단어들이 변경, 추가, 제거되어 무의미하고 부적절한 라틴어로 변형되었습니다. "Lorem ipsum"의 처음 두 단어는 "dolorem ipsum"("고통 그 자체")의 축약입니다.

이 텍스트의 사용은 1960년대부터 시작되었으며, Letraset 전사 시트 광고에서 인기를 얻었습니다. 1980년대 중반, Aldus Corporation이 데스크톱 출판 프로그램 PageMaker의 템플릿에 이 텍스트를 도입하면서 디지털 세계에 본격적으로 소개되었습니다. 이후 Apple의 Pages, Microsoft Word와 같은 워드 프로세서, Joomla!와 WordPress 같은 웹 콘텐츠 관리 시스템, Semantic UI 같은 CSS 라이브러리에서도 채택되었습니다.
